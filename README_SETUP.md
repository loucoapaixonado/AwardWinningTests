# Como executar (local)

- Abra `index.html` no navegador (arraste para o navegador, ou use Live Server/servir arquivos estáticos).
- Projeto é um SPA simples feito com HTML/CSS/JS (módulos ES).
- Dados de exemplo estão em `data/questions.js` e `data/cards.js`.
- A coleção é guardada em `localStorage` (chave `award_cards_collection_v1`).
- O histórico de perguntas já feitas é salvo em `localStorage` (chave `award_cards_asked_v1`) para evitar perguntas repetidas.

Pontos a ajustar ou expandir
- Adicionar imagens reais no diretório `assets/` ou substituir por URLs.
- Incluir testes e otimizações de acessibilidade.

Feito para uso educativo e mobile-first. Boa exploração!
